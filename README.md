
# DroidClaw

在 Android 上运行的 OpenClaw AI 网关，独立运行，无需 Termux。

## 下载

- 本仓库**不包含软件源代码**，仅提供**安装包（APK）** 下载。
- 请在 [Releases](https://github.com/iskandar510/DroidClaw/releases) 页面下载最新版本 APK 并安装。

详见 [使用说明](docs/使用说明.md)。

## 基于原版的修改说明

详见 [修改说明](docs/修改说明.md)。

## 国内网络说明

原版依赖在线下载全部环境与依赖，部分资源（尤其是 OpenClaw 本体）在国内难以下载。本版本通过**依赖镜像**和**将 OpenClaw 完整编译内容打包进 APK** 的方式解决该问题。详见 [国内下载问题说明](docs/国内下载问题说明.md)。

## 开发与二次开发

- 本仓库为**闭源项目**，此处仅发布安装包。
- 若您基于其他开源项目进行二次开发，请**必须保留并声明原作者信息**，感谢开源作者。详见 [开发文档](docs/开发文档.md)。
