# 在原有基础上的修改说明

本软件在 [openclaw-termux](https://github.com/mithun50/openclaw-termux) 等原有项目基础上进行了以下修改与增强。

## 1. 界面与文案汉化
- 应用内界面、设置项、提示信息等改为中文，便于国内用户使用。

## 2. 终端常用命令列表（可自定义）
- **内置常用命令**：在终端界面提供一组与 OpenClaw 相关的快捷命令（如网关模式、Token、启动/停止/重启、配置查看与设置、模型/渠道管理、状态与诊断、日志、版本与更新等），点击即可填入终端并执行。
- **自定义命令**：支持用户自行添加、编辑、删除「显示名称 + 实际命令」，数据持久化保存，与内置命令一起在终端中展示和使用。

## 3. 修改 OpenClaw 配置文件
- 新增**配置编辑器**界面，可直接查看和编辑 `root/.openclaw/openclaw.json`。
- 支持 JSON 格式校验，避免保存非法配置；保存后写入 rootfs 内对应路径。

## 4. 设置网关 Token 的选项
- 在**设置**中增加「网关令牌」入口。
- 用户可查看、设置或重置用于访问 Web 仪表板的安全令牌；未设置时使用随机生成的令牌。

## 5. 国内网络与依赖
- 在依赖安装阶段使用 **npm 国内镜像**（如 npmmirror.com），在无法使用内置 OpenClaw 时从镜像安装。
- 将 **OpenClaw 完整编译产物** 打包为 `openclaw.tgz` 放入 APK 资源，首次安装优先从资源解压安装，减少国内无法下载本体的问题。详见「国内下载问题说明」。
